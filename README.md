# ゆーかねすぴれこ(Yukarinette Speech Recognition)

## 概要
ゆかりねっと-Yukarinette-の音声認識エンジンを別処理で認識させるためのツールです。  
[配布ページはこちら](https://harukei66494739.github.io/recognize/)


## 自分でビルドしたい
build-all.batを実行してexeを作成できます。  


## 規約
